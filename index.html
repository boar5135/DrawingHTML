Components of the d3 Algorithm

.Select () .Append () .Text () .Style ()
When using .select () in javascript, the CSS selector that follows it will identify the part of the document that is to be used in the following code. With the .append () in javascript, it creates the element that one chooses to add and adds it to the end of whatever part of the document was selected. Using the .text () in javascript will insert text in between the tags of the part that was previously selected. With the .style () in javascript, it alters the HTML using CSS properties.
